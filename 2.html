Introdução GIT

  • cd web - entra na pasta web
  • code ./ - comando que abre o VS Code
  • git init - cria um repositório em um diretório
  • git add - adicionar modificações do workcopy worktree ao stage
  • git commit -m "asdf" - cria um commit a partir do stage
  • git log - exibe os commits do nosso repósitorio local
  • git log --oneline --all --graph
  • git status - exibe o estado do WK
  • git checkout -b nova-branch branch|hash - cria uma nova branch apontando para a branch ou hash informado
  • git reset --hard - restaura os arquivos trakeados ao estado do HEAD
  • git push remote branch
  • git config user.name = "Nome"
  • git config user.email = "Email
  • git clone url - faz o clone de um repositório
