Projeto – Atividades de JavaScript
Este repositório reúne 12 atividades práticas de JavaScript, desenvolvidas com foco na aprendizagem dos principais conceitos da linguagem.
Cada exercício possui sua própria página HTML, onde a lógica foi implementada usando JavaScript puro.

- LISTA DE ATIVIDADES

-> Att 1 – O que é JavaScript?
Introdução ao JavaScript: conceito, uso e funcionamento dentro de páginas web.

-> Att 2 – Tipos de Variáveis e Operadores
Exploração dos tipos primitivos (string, number, boolean etc.) e operadores aritméticos e de comparação.

-> Att 3 – Estruturas de Seleção (IMC com if e switch)
Cálculo do IMC usando estruturas condicionais.
Duas versões: com if/else e com switch.

-> Att 4 – Gerador de Números Aleatórios com Intervalo
Usuário informa um intervalo, e o script gera um número aleatório dentro dele usando Math.random().

-> Att 5 – Números Primos em um Intervalo
Verificação e listagem de todos os números primos dentro de um intervalo definido.

-> Att 6 – Verificador de Palíndromos
Verifica se uma palavra ou frase é um palíndromo (lida igual de trás para frente).

-> Att 7 – Mudança de Cor de Fundo com Clique
Ao clicar na tela, a cor do fundo é alterada dinamicamente.

-> Att 8 – Alteração de Cores com Tecla de Espaço
A cor do fundo muda sempre que o usuário pressiona a barra de espaço.

-> Att 9 – Calculadora Aritmética
Pequena calculadora que realiza as quatro operações básicas: adição, subtração, multiplicação e divisão.

-> Att 10 – Explorando Eventos em JavaScript
Exercício focado em eventos como: click, keydown, focus, blur, mouseover, mouseout e change.

-> Att 11 – Salvando Dados no Local Storage
Salva e recupera o nome do usuário usando localStorage.

-> Att 12 – Desenhando com Canvas
Desenho de elementos gráficos com Canvas 2D: retângulo, círculo e texto.
